# Real-Time Channel-Based Chat Application

A full-stack real-time chat application where users can create channels, join channels, and exchange messages instantly.  
The system supports JWT authentication, MongoDB persistence, and live messaging using Socket.io.

### Core Features
- User authentication (Signup & Login) with JWT
- Create and join channels
- Real-time messaging inside channels
- Persistent chat history stored in MongoDB
- Socket.io-based communication
- Secure APIs (protected routes)

### Optional Features Implemented
- Display sender names with messages
- Auto-scroll for messages
- Real-time online users name.

---

## Tech Stack

### **Frontend**
- React.js
- Context API for state management
- Socket.io-client
- TailwindCSS && CSS
- Notistack for notifications

### **Backend**
- Node.js + Express.js
- MongoDB + Mongoose
- Socket.io (WebSockets)
- JWT Authentication
- CORS
